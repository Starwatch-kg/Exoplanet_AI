# 🌌 Exoplanet AI - Веб-платформа для поиска экзопланет

## 📋 Обзор проекта

Создана полнофункциональная веб-платформа для поиска экзопланет с использованием искусственного интеллекта. Проект включает современный React frontend с интерактивными графиками и FastAPI backend, интегрированный с существующим ML пайплайном.

## ✅ Выполненные задачи

### 1. Структура проекта ✅
- Создана архитектура frontend + backend
- Настроена интеграция с существующим ML кодом
- Организованы скрипты запуска

### 2. Frontend (React + Vite + Tailwind) ✅
- **Современный UI**: Космическая тема с анимациями
- **Компоненты**: Dashboard, DataLoader, ModelSelector, Results
- **Интерактивные графики**: Plotly.js с зумом и наведением
- **Адаптивный дизайн**: Работает на всех устройствах
- **TypeScript**: Полная типизация

### 3. Backend (FastAPI) ✅
- **REST API**: Полный набор эндпоинтов
- **Интеграция с ML**: Подключение к существующему пайплайну
- **CORS**: Настроен для работы с frontend
- **Обработка ошибок**: Корректная обработка исключений

### 4. Интерактивные возможности ✅
- **Анимации**: Framer Motion для плавных переходов
- **Звездное небо**: Анимированный фон с Canvas
- **Графики**: Интерактивные кривые блеска, периодограммы
- **Реальное время**: Статус анализа в процессе

### 5. Модели ИИ ✅
- **Autoencoder**: Детекция аномалий
- **Classifier**: Бинарная классификация
- **Hybrid**: BLS + нейронные сети
- **Ensemble**: Ансамбль всех моделей

## 📊 API Endpoints

- `GET /` - Информация о API
- `GET /health` - Проверка состояния
- `GET /models` - Список доступных моделей
- `POST /load-tic` - Загрузка данных TESS
- `POST /analyze` - Анализ кривой блеска
- `GET /results/{tic_id}` - Получение результатов
- `DELETE /results/{tic_id}` - Очистка результатов
